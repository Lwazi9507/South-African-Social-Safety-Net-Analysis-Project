    # 🇿🇦 South Africa Social Safety Net Analysis (Power BI Project)

## 📊 Project Overview
This project analyzes the growth and distribution of South Africa’s social safety net programs using data inspired by World Bank and CEIC statistics.

The analysis focuses on:
- National coverage expansion
- Social protection program types
- Equity and wealth distribution
- Coverage across income quintiles

The goal is to transform raw socio-economic data into actionable insights using **Power BI**.

---

## 🎯 Objectives
- Clean and structure social protection data
- Analyze population coverage trends
- Compare safety nets vs social insurance
- Visualize wealth-based distribution equity
- Build an interactive Power BI dashboard

---

## 🧰 Tools & Technologies
- Microsoft Power BI (.pbix)
- Excel / CSV Data Sources
- Data Modeling (Power Query)
- DAX Measures
- Data Visualization

---

## 📁 Dataset Description

| Column | Description |
|------|-------------|
| Year | Reporting year |
| Program_Type | Type of social support |
| Coverage_Percentage | Population coverage |
| Wealth_Quintile | Income group classification |
| Population_Group | Target beneficiaries |

Programs included:
- Cash Transfers
- School Feeding
- Food Vouchers
- Disability Benefits
- Public Works

### 2️⃣ Social Insurance vs Safety Nets
- Coverage comparison visuals

### 3️⃣ Equity & Wealth Distribution

## 📊 Key Insights
- Social safety nets reached over **60% population coverage**
- Highest impact observed among the **poorest 20%**
- Coverage declines progressively with wealth increase
- Targeted distribution improves equity outcomes
